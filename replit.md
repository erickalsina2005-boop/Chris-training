# Chris Training - Landing Page

## Overview
A static HTML landing page for "Chris Training" - a fitness coaching program. The page is in Spanish and promotes a 6-week body transformation program.

## Project Structure
- `chris-training-2.html` - The single-page HTML landing page (all CSS and JS inline)

## Running the Project
The project is served via Python's built-in HTTP server:
- **Command:** `python3 -m http.server 5000`
- **Port:** 5000

## Deployment
Configured as a static site deployment with the root directory (`.`) as the public directory.
